LightHouse role
=========

The role install and configure NGINX and LightHouse.

Requirements
------------

- Centos  == 7
- Ansible >= 2.10

Dependencies
------------

- epel-release

Example Playbook
----------------

```yaml
  - hosts: clickhouse
    roles:
      - lighthouse-role
```
